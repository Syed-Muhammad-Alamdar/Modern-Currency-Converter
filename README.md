# 💱 Modern Currency Converter

A modern Currency Converter web application built using **HTML**, **CSS**, and **JavaScript**. It fetches live exchange rates through a REST API and instantly converts currencies with a clean and interactive user interface.

---

## 🚀 Features

- 🌍 Real-time currency conversion
- 🔄 Live exchange rates using Fetch API
- ⚡ Asynchronous JavaScript (Async/Await)
- 🚩 Dynamic country flag updates
- 💱 Supports multiple international currencies
- ✅ Input validation
- 🎨 Modern responsive UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- JSON
- REST API
- DOM Manipulation

---

## 📂 Project Structure

```text
index.html    → Main application

style.css     → Styling

app.js        → API requests & conversion logic

Codes.js      → Currency and country code mapping
```
